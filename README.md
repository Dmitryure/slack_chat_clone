
## *Для начала работы*
  `git clone git@github.com:Dmitryure/slack_chat_clone.git`
  
`cd slack_chat_clone/chb` 

`npm i`

 `npm start`
 

## Функционал
1) Регистрация/логин
2) Создание каналов
3) Функции чата
4) Добавление каналов в избранное
5) Изменение стиля
6) Прочие фичи
## Инструменты
Backend: Firebase

Frontend: React, Redux, semantic-ui-react
